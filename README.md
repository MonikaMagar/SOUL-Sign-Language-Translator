**SOUL** is a real-time **Sign Language Recognition and Translation System** that enables communication for people relying on sign language. The system captures hand gestures using a camera and sensors, processes the input using advanced algorithms, and converts recognized gestures into meaningful outputs.

- **Accessibility:** Improves communication for people using sign language.
- **User-Friendly Interface:** Responsive and intuitive UI built with React.
- **API Integration:** Optional SignAll API for enhanced gesture recognition.
